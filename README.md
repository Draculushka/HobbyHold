# HobbyHold

HobbyHold — это платформа для публикации и обмена хобби. Проект представляет собой полноценное веб-приложение, построенное на современном стеке технологий с использованием многослойной архитектуры (Layered Architecture).

---

## Основные возможности

*   **Web UI:** Современный интерфейс на Tailwind CSS для просмотра, создания и поиска хобби.
*   **Многослойная архитектура:** Четкое разделение ответственности между API, бизнес-логикой и настройками.
*   **Поиск по синонимам:** Умный поиск, понимающий разные названия одного и того же увлечения.
*   **Безопасность:** JWT-авторизация (HttpOnly Cookies), хеширование паролей и очистка HTML (nh3).
*   **Загрузка изображений:** Поддержка прикрепления фотографий к постам с автоматической очисткой старых файлов.
*   **Docker Ready:** Запуск всей инфраструктуры одной командой через Docker Compose.

---

## Стек технологий

*   **Backend:** FastAPI (Python 3.12)
*   **Database:** PostgreSQL + SQLAlchemy (ORM)
*   **Templates:** Jinja2 + Tailwind CSS
*   **Testing:** Pytest + HTTPX
*   **Infrastructure:** Docker, Docker Compose, Nginx, Redis, MinIO
*   **Migrations:** Alembic

---

## Архитектура проекта (Layered Architecture)

Проект использует модульную структуру для обеспечения масштабируемости и тестируемости:

*   **`api/`**: Слой обработки HTTP-запросов (маршруты).
*   **`services/`**: Слой бизнес-логики (сохранение данных, обработка изображений, теги).
*   **`core/`**: Глобальные настройки, конфигурация безопасности и JWT.
*   **`models/`**: Описание структуры таблиц БД.
*   **`tests/`**: Набор автоматизированных тестов (Unit и Integration).

```text
hobbyhold/
├── api/             # Эндпоинты (auth, hobbies)
├── services/        # Бизнес-логика
├── core/            # Настройки и безопасность
├── main.py          # Точка входа (сборка приложения)
├── models/          # Модели БД
├── database.py      # Подключение к БД
├── templates/       # HTML шаблоны
├── static/          # Статические файлы (CSS, JS)
└── tests/           # Автоматизированные тесты (pytest)
```
